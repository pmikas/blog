Django

               Here you can find some solutions for problems that I encounter when I was creating this blog.

Creating a project:
  1. Create a folder for the website
  2. Open the cmd prompt
  3. Type: django-admin startproject 'project_name'
  4. Go to the folder that was created: cd 'project_name'
  5. Try to run the server, typing: python manage.py runserver

Sending code to Github at cmd:
  1. git status
  2. git add --all
  3. git status
  4. git commit -m "Comments"
  5. git push

Update static files in PythonAnywhere
python3 manage.py collectstatic

Whenever a new model is made:
  1. python manage.py makemigrations
  2. python manage.py migrate

Git syncronization in PythonAnywhere
  1. git reset --hard HEAD
  2. git merge upstream/master
  3. git pull

"please enter a commit message to explain why this merge is necessary, especially if it merges an updated upstream into a topic branch."
  1. press i
  2. write message
  3. press esc
  4. write :wq
  5. press enter
